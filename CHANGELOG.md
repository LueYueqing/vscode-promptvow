# PromptVow VSCode Extension Changelog

## [0.1.2] - 2026-01-18

### 新增功能
- ✨ 双击提示词自动复制内容到剪贴板
- ✨ 右键菜单新增"完成任务"选项，可将提示词状态设置为 COMPLETED
- ✨ 在视图中直接集成操作按钮，提升用户体验
- ✨ 新增提示词按钮集成到视图底部，方便快速添加

### 优化改进
- 🎨 视图UI全面优化，操作按钮直接显示在左侧视图中
- 🎨 未选中项目时显示明显的"选择项目"按钮
- 🎨 选中项目后显示"当前项目"信息，点击可切换
- 🎨 空状态显示友好的提示信息和操作引导

### Bug修复
- 🐛 修复 tags 字段类型错误（从数组改为字符串格式以匹配API）
- 🐛 修复 API 客户端中 tags.join() 类型错误
- 🐛 修复 saveSelectionAsPrompt 函数中的 tags 参数错误

### 技术改进
- 📦 优化 ProjectProvider，支持操作按钮自动执行命令
- 📦 改进 ProjectTreeItem，支持命令绑定和图标配置
- 📦 完善错误处理和用户提示信息

### 用户体验提升
- 用户无需通过顶部工具栏即可完成常用操作
- 操作流程更清晰：选择项目 → 查看提示词 → 新增/操作
- 保留顶部工具栏和右键菜单，满足不同用户习惯

---

## [0.1.1] - 2026-01-18

### 初始版本
- 基础的项目和提示词管理功能
- 支持选择项目和查看提示词列表
- 右键菜单支持复制提示词和设置默认项目
- 支持保存选中文本为提示词
- 支持搜索提示词功能
